### 账号密码登录{#loginByPwd}

```
POST /user/loginByPwd
```
### 字段说明

| 参数名         | 类型            | 描述                          | 是否必须            |
| ----------- | ------------- | --------------------------- | --------------- |
| account          | string   | 账号                     | 是               |
| password          | string   | 密码                     | 是               |
