## 游戏 js 调用Android 方法

js方法
```
var game={
    "action":1,
    "jumpUrl":"",
    "msg":""
}
game为String类型
window.android.callAndroid(game);
```
```js
action==1时，全屏视频广告播放,action==2时， 触发底部banner广告,

```
